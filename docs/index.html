Mirror Engine

Website capture with guardrails, previews, and exportable outputs.

This GitHub Pages site is a public launchpad for Mirror Engine. The full product is a server-hosted Next.js application with API routes, Playwright capture, and persistent clone storage, so the complete runtime does not fit on GitHub Pages.

Why Pages is only the static front door

  • GitHub Pages can serve the public landing content.
  • The full app still needs a real server for API routes and Playwright.
  • The long-term production target should be a container-hosted Next.js deploy.
